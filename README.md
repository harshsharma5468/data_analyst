

A comprehensive Streamlit application for data analysis, visualization, and machine learning.

## Features

- **Data Upload**: Support for CSV, Excel, and Parquet files with automatic encoding detection
- **Exploratory Data Analysis (EDA)**: Summary statistics, missing values analysis, correlations, outlier detection
- **Data Profiling**: Comprehensive data profiling with ydata-profiling
- **Visualization**: Quick charts, interactive Plotly charts, and custom chart builder
- **SQL Lab**: Execute SQL-like queries on your data
- **Machine Learning**: Classification and regression with multiple algorithms, cross-validation, and hyperparameter tuning
- **Time Series Analysis**: Decomposition, SARIMAX modeling, and forecasting
- **Reports**: Generate and export analysis reports
